## Install

1. git clone -b front-end --single-branch https://github.com/bradcodd/thumio-training-day-3.git thumio-training-day-4
2. cd thumio-training-day-4
3. copy ..\thumio-training-day-3\.env
4. rd /s .git
5. git init
6. git config --global --add safe.directory "E:/Projects/Vibe Coding/thumio-training-day-4"
7. git add .
8. git status
9. git commit -m "Cloned from thumio-training-day-3/front-end"
10. git branch -M main
11. git remote add origin https://github.com/bradcodd/thumio-training-day-4.git
12. git push -u origin main
13. npm install
14. npm run dev
